Open-Shell

Что такое Open-Shell?

Open-Shell™ представляет собой коллекцию улучшений для Windows. Open-Shell имеет настраиваемые главное меню и кнопку Пуск, добавляет панель инструментов для проводника Windows и поддерживает множество мелких функций.

Системные требования

Open-Shell работает на Windows 7, Windows 8, Windows 8.1, Windows 10 и Windows 11, вместе со своими серверными аналогами. Поддерживает 32 и 64-разрядные версии (универсальная программа установки). Для некоторых обложек меню "Пуск" необходимо включить тему Aero. Для других обложек требуется тема упрощенного стиля Windows.

Компоненты

Open-Shell has three major components:
  • Меню Open-Shell
  • Classic Explorer
  • Classic IE

Удаление

Вы можете удалить Open-Shell через Панель управления -> Программы и компоненты. Другой способ удаления заключается в  в повторном запуске программы установки и выборе команды "Удалить".
Возможно потребуется завершить сеанс для завершения этого процесса.
